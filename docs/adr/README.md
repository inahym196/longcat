```md
# nnn: Title
## Status
<!-- Proposed | Accepted | Superseded | Rejected -->

## Context
<!-- 
- 現在の状況
- なぜ検討が必要か
- 制約条件
-->

## Decision
<!-- 
- 採用する選択

-->

## Consequences

### Positive
<!-- 
- 利点
- 期待効果
-->

### Negative
<!-- 
- 制約
- 将来コスト
-->

## Alternative Considered
<!-- 検討した他案と却下理由 -->
```
